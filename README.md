# 🧪 SauceDemo Automation Test Suite

## ✅ Key Features Covered
Login functionality

Add to cart

Checkout flow

Confirmation page validation

Page Object Model (POM) using PageFactory

Cucumber BDD scenarios and step definitions

Assertions with TestNG

Scalable folder structure

Easily extendable and maintainable

## 🧰 Tech Stack
Layer	Tools/Frameworks Used
Language	Java 17
Test Framework	Cucumber + TestNG
Automation Library	Selenium WebDriver 4.7.2
Build Tool	Maven
Browser Drivers	WebDriverManager
Reporting	Allure (optional, included in dependencies)
Utilities	PageFactory, Waits, Screenshots, etc.
